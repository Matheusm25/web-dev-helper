# Command Reference for wdh

## Get rgb and rgba from hexadecimal color
```shell
$ wdh hex-to-rgb 878787
Your rgb color: rgb(135, 135, 135) or rgba(135, 135, 135, 1)
```
or with the alias:
```shell
$ wdh htr 878787
Your rgb color: rgb(135, 135, 135) or rgba(135, 135, 135, 1)
```

### Generate random string
```shell
$ wdh random --string
```
or with the alias:
```shell
$wdh rdm --string
```
